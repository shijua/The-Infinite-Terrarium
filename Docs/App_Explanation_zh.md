# The Infinite Terrarium 文字稿（中文）

## 一句话介绍
`The Infinite Terrarium` 是一个离线运行的 AI 数字生态缸：你可以实时观察上千个数字生物的群体行为，并通过 `Feed / Mutate / Analyze` 三个动作干预生态与理解变化。

## 作品在做什么
这个 App 把三个系统放在一起：
1. `Simulation`：使用 Boids + Quadtree 的实时群体仿真，驱动“生物”移动、聚散、能量变化和种群兴衰。
2. `Rendering`：用 SwiftUI Canvas + Metal Shader 做液态、折射、粒子背景等视觉效果，并按性能自动调节画质。
3. `AI`：调用端侧 Foundation Model，为当前生态做解释，并生成可注入的新物种参数。

## 用户操作与结果
1. `Feed`：在局部区域注入能量脉冲，附近生物会短时恢复活性，群体轨迹会更稳定。
2. `Mutate`：对主导物种做参数突变（社交距离、对齐、凝聚、代谢、速度），会直接改变后续运动形态和生存压力。
3. `Analyze`：根据当前快照生成解释文本，回答“为什么某类生物在减少/扩张”。
4. `Inject Species`：让 AI 生成一组新 DNA 并注入生态，观察新物种与旧群落的竞争和共存。

## 你在画面里看到的含义
1. 左上角 `FPS/Sim/Render`：实时性能指标，反映仿真与渲染耗时。
2. `Population / Avg Energy / At Risk`：生态健康度指标。
3. `Quality`：当前画质档位（HIGH/MEDIUM/LOW），系统会根据帧预算自动切换。
4. 各颜色生物：对应不同物种的 `hue` 与行为参数组合，不同颜色不只是外观差异，而是行为策略差异。
